Stub

Stub

Class that creates a stub method on an object, and provides useful methods to track the response returned on each call.

This class is not meant to be instantiated directly, but is designed for use within the ObjectMock class.

new Stub(instance, methodName, returnValue)

Parameters:
Name Type Description
instance Object The object instance on which the method will be stubbed.
methodName String The name of the method on the object that needs to be stubbed. If the specified method does not exist, a placeholder method will be injected into the instance and then stubbed.
returnValue * | function A return value that will be returned by the stub. If a function is passed in, the function will be invoked, and ther return value of the function will be returned as the response.
